Stack Sorting
by CodeChum Admin

Given a stack containing integers, write a function to sort the stack in ascending order using another stack as an auxiliary data structure. You are not allowed to use any other data structures such as arrays or linekd lists.

Enter element 1: 5
Enter element 2: 9
Enter element 3: 7
Enter element 4: 6
Enter element 5: 5
Sorted Stack: 9 7 6 5 5
