[E1] - Music Playlist - CURSOR BASED
by Edrian Guanzon

Music playlists are a common feature in many digital platforms, allowing users to organize and manage their favorite songs. This application simulates a simple playlist system where songs can be added, removed, and displayed in a specific order. Since the playlist has a limited storage capacity, it also demonstrates how data can be efficiently managed within a fixed amount of memory.

 

Objective

Must provide the following options:
Add Song – add new song to the list
Delete Song – delete the song from the list based on the name
Display List – show all songs in the list with artist name
Exit – terminate the program
 

Function Definition

insertEnd() -> inserts a new song to the list
deleteSong() -> deletes the song from the list based on the song title
